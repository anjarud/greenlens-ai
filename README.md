# GreenLens AI

GreenLens AI is a full-stack web application for AI-assisted plant image analysis.

Users can upload plant images, create analysis jobs and receive normalized plant identification results from a separate AI service connected to the Pl@ntNet REST API.

## Tech Stack

- Frontend: React, Vite, TypeScript
- Backend: Python, FastAPI
- AI Service: Python, FastAPI
- External API: Pl@ntNet REST API
- HTTP Client: httpx
